What you need

In order to develop with NORT, you will need:

  • A text editor. (Personally I like VSCode)
  • A web server. There are multiple options.

    If you are building a complete system that includes an application server, just place your NORT files in the static files folder of your application server.

    To service only you UI files, a simple static files server will do. Multiple options exist; here are two easy ones:

    • Python3. Install python3. From the root folder where you placed you NORT files, run python -m http.server 8000
    • PHP-CLI. Install PHP. From the root folder where you placed you NORT files, run php -S 0.0.0.0:8000

    Your application is accessible at http://localhost:8000.

  • The NORT library. See below options for obtaining the library
    • For development, you may simply include the NORT system by including the github version in your HEAD section, like in this example.

Including the NORT library

NORT is fully compliant with ECMAscript 2020 modules.

The best method is to get a static version of NORT, and add it to your application assets. You can download it from https://stefpo.github.io/nort/dist/nort.tgz or https://stefpo.github.io/nort/dist/nort.zip
You can then import it in your application code. 


    import * as nort from "./nort/nort.js"

Make sure you also import the other modules you needs. (See API)
